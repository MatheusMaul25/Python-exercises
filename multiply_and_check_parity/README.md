Multiplicação e Verificação de Paridade

Este projeto é um exercício simples em Python que utiliza funções para multiplicar números e verificar se o resultado é par ou ímpar.

Como funciona

O programa possui duas funções:

multiplicacao(*args)
Recebe vários números como argumentos e retorna o resultado da multiplicação entre eles.

impar_par(numero)
Verifica se um número é par ou ímpar.

Após realizar a multiplicação, o programa mostra o resultado e informa se o número é par ou ímpar.

Exemplo de execução
multiplicar = multiplicacao(10, 20, 30, 40, 50)

print(multiplicar)
print(f'O número {multiplicar} é', impar_par(multiplicar))

Saída no terminal:

12000000
O número 12000000 é Par
