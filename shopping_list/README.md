Shopping List (Lista de Compras em Python)

Este projeto é uma lista de compras simples em Python, criada para praticar conceitos básicos de programação e organização de código.

O programa permite adicionar, visualizar e apagar itens em uma lista de compras utilizando o terminal.

Como funciona

O programa executa no terminal e permite interagir com uma lista de compras.
Os itens adicionados são armazenados em uma lista durante a execução do programa.

Como executar

Execute o arquivo shop_list_main.py:

python shop_list_main.py

O programa será iniciado no terminal e permitirá adicionar, apagar e listar itens à lista de compras.
