Gerador de CPF em Python

Este projeto é um gerador de números de CPF válidos, desenvolvido em Python para fins de estudo e prática de programação.

O programa gera os 9 primeiros dígitos aleatórios e calcula os 2 dígitos verificadores utilizando o algoritmo oficial do CPF.

1- Como funciona

O programa gera 9 dígitos aleatórios.

Calcula o primeiro dígito verificador usando o algoritmo do CPF.

Calcula o segundo dígito verificador.

Retorna um CPF válido com 11 dígitos.

2- Como executar

Execute o arquivo gen_cpf_main.py:

python gen_cpf_main.py

O programa irá pedir a quantidade de CPF desejada e gerar o número desejado de CPF's
